Sorting storage can be a pain when you’ve got items spread across loads of chests. With this mod, you simply select the chests you want, each one clearly outlined so you can see what’s included, and it automatically sorts and distributes items between them based on quantity.

----------------------------------------------------------------------------------------------

The default key to select a chest is #.
Once you have selected a chest, or multiple, simply press F7 to sort the items by quantity.
The items will automatically be put into the chest you selected first, then second, and so on.
